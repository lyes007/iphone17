# iphone17

iPhone 17 Product Display Website

A Next.js application showcasing the iPhone 17 with a beautiful liquid glass loading animation and modern UI.

## Features

- Liquid glass loading animation with logo
- Responsive navbar with glassmorphism effect
- Smooth fade-in animations
- Mobile-optimized layout

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- @paper-design/shaders-react

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
